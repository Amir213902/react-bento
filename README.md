# Тестовое задание

## Технический стэк:
- React
- React-Router-Dom
- React-grid-layout
- React-icons

## Что нужно чтобы запустить?
- Установленный на вашем компьютере Node Js
- Система контроля версией git (необезательно, в таком случае можете загрузить архивом проект)
- Выбранная/созданная вами папка для хранения проекта

## Как запустить?
После загрузки проекта в выбранной вами папку, следует в терминале прописать две команды поочередно:
```npm i```, а затем ```npm run dev```. Чтобы увидеть запущенную вами программу введите в браузере те адрес и порт, что были отабражены в терминале после запуска.
