# Тестовое задание

## Технический стэк:
- React
- React-Router-Dom
- React-grid-layout
- React-icons

## Что нужно чтобы запустить?
- Установленный на вашем компьютере Node Js
- Система контроля версией git (необезательно, в таком случае можете загрузить архивом проект)
- Выбранная/созданная вами папка для хранения проекта

## Как запустить?
После загрузки проекта в выбранной вами папку, следует в терминале прописать две команды поочередно:
```npm i```, а затем ```npm run dev```. Чтобы увидеть запущенную вами программу введите в браузере те адрес и порт, что были отабражены в терминале после запуска.

## Как добавить еще уникальные типы блоков?
По пути ```./src/constants/constsBlocks.jsx``` указываются все возможные типы блоков (от youtube до мессенджера X). Добавьте там свои уникальные блоки, ориентируясь на другие блоки.

## Какие еще фишки здесь обитают?
Можете в пропсах с префиксом background потставить не только сплошные цвета (rgb, rgba, hex), но также градиент потставив в кавычки css свойство ```liner-gradient(угол поворота, цвет1, цвет2...
цветN)```.

## Как подобрать градиент?
Чтобы решить такой вопрос обратитесь к бесплатным онлайн сервисам вбив в поисковую систему ```градиент css```. Обычно при копировании готовый градиент css идут два стиля ```background: hex или rgba``` и ```background: liner-gradient(параметры)```. В таком случае в файле вы прописываете в свойствах нового уникального типа блока ```'(hex, rgba), liner-gradient(параметры)'```. В крайнем случае если ничего не поняли попробуйте поиграться с типом ```instagramBlock```, в котором уже прописаны некоторые градиенты и посмотрите на результат.
